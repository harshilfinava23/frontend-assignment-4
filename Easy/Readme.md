# Bento Grid UI

A bento-style photo grid layout built with pure **HTML & CSS**.  
This project focuses on understanding **CSS Grid Layout** with overlapping image effects.

🔗 **Live Demo:** [grid-layout-ui.vercel.app/Easy](https://grid-layout-ui.vercel.app/Easy)

## About

The goal was to recreate a given bento grid UI design as close as possible using:

- `display: grid`
- `grid-template-columns`
- `grid-template-rows`
- `grid-template-areas`
- `gap`
- `position: relative`
- `position: absolute`

The center model image overflows its grid cell using `position: absolute` + `height: 150%` for a layered visual effect.

No JavaScript. No frameworks. Pure HTML & CSS only.
