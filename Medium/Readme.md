# CSS Grid Layout UI

A responsive dashboard UI built with pure **HTML & CSS**.  
This project focuses on understanding **CSS Grid Layout** in real UI layouts.

🔗 **Live Demo:** [grid-layout-ui.vercel.app](https://grid-layout-ui.vercel.app)

## About

The goal was to recreate 3 given UI designs as close as possible using:

- `display: grid`
- `grid-template-columns`
- `grid-template-rows`
- `grid-template-areas`
- `gap`
- `position: relative`
- `position: absolute`

No JavaScript. No frameworks. Pure HTML & CSS only.
